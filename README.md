# Webpack

概念：是一个JavaScript应用程序的静态模块打包器
作用：将项目的资源文件当成一个个模块，模块之间有依赖关系，webpack会对这些有依赖关系的文件进行处理，让浏览器能够识别，最后将应用程序需要的每个模块打包成一个或多个bundle。
